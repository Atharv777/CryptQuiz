## Getting Started

Follow the below steps to get the project working.

1. Install the dependencies:

   ```sh
   # Using NPM
   npm install

   # Using Yarn
   yarn
   ```

2. Run using:

   ```sh
   # Using NPM
   npm start

   # Using Yarn
   yarn start
   ```
